## 📁 模板清單

### 1. kanban-board.json
**用途：** 個人項目管理  
**章節：** Chapter 4  
**如何使用：**
```bash
# 複製到 workspace
cp kanban-board.json /your/workspace/

# 在瀏覽器打開
# https://kanban-board.zeabur.app
```

### 2. HEARTBEAT.md
**用途：** 每日自動化任務

**如何使用：**


### 3. MEMORY.md
