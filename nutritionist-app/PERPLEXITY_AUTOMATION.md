# 🤖 Perplexity Pro 瀏覽器自動化指南

**目標：** 用 Perplexity Pro 分析食物圖片 + 提取營養分析

---

## 📋 步驟

### 1️⃣ 準備工作

**需要：**
- ✅ Perplexity Pro 賬戶（你已經有！）
- ✅ Chrome 瀏覽器
- ✅ OpenClaw Chrome 擴展

### 2️⃣ 連接瀏覽器

**步驟：**

1. **打開 Chrome 瀏覽器**
   - 網址：https://www.perplexity.ai
   - 確保已登入 Pro 賬戶

2. **點擊 OpenClaw Chrome 擴展圖標**
   - 喺 Chrome 工具列（右上角）
   - 搵到 OpenClaw 圖標
   - 點擊佢

3. **啟用瀏覽器 Relay**
   - 擴展會顯示「Attach Tab」或類似提示
   - 點擊確認
   - 等待連接成功

### 3️⃣ 測試自動化

**連接成功後，我可以幫你：**

1. 打開 Perplexity
2. 上傳圖片
3. 發送問題：「請分析呢張圖片中的食物，並提供營養成份（卡路里、蛋白質、脂肪、碳水化合物）」
4. 等待 AI 回答
5. 提取並保存答案

---

## 🎯 自動化流程

```
用戶指令 → Jarvis → 瀏覽器自動化 → Perplexity → 提取答案 → 保存結果
```

**優勢：**
- ✅ 用你已有的 Pro 賬戶（唔使額外付費）
- ✅ Perplexity 識識別圖片（用 GPT-4V/Claude）
- ✅ 準確度高
- ✅ 支援中文/廣東話

**限制：**
- ⚠️ 需要手動連接瀏覽器（每次）
- ⚠️ 速度較慢（10-30 秒/次）
- ⚠️ 需要網絡連接
