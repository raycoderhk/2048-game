# 🎮 2048 遊戲

經典 2048 遊戲 - 合併數字到 2048 就贏咗！

## 🎯 遊戲說明

- 使用 **方向鍵** 或 **滑動** 控制方塊移動
- 相同數字碰撞時會 **合併**
- 合併到 **2048** 就贏咗！
- 無處可移時遊戲結束
- 按 **撤銷** 可以悔棋一次

## 🕹️ 控制方式

### Desktop
- **↑** 向上移動
- **↓** 向下移動
- **←** 向左移動
- **→** 向右移動

### Mobile
- **滑動** 上下左右

## 🏆 技巧貼士

1. **角落策略** - 將最大數字keep喺角落
2. **唔好亂撳** - 每個移動都要有目的
3. **預先計劃** - 諗住下一步點行
4. **適時撤銷** - 行錯咗即刻 undo
5. **保持冷靜** - 越急越容易錯
