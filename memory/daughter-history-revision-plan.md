## 📝 Revision Techniques / 溫習技巧

### 1. Flashcards / 閃卡
- Front: Event/Person / 事件/人物
- Back: Date/Significance / 日期/重要性

### 2. Timeline Method / 時間線法
```
History: Event A → Event B → Event C
中史：朝代 A → 朝代 B → 朝代 C
```

### 3. Mind Maps / 思維導圖
- Central topic in middle / 中心主題在中間
- Branch out key points / 分支出重點

### 4. Practice Questions / 練習題目
- Source-based questions / 資料為本題
- Essay outlines / 論文大綱
